STI College Novaliches Online Timekeeping Authorization Manager

# Installation
1. Install Composer dependencies (`composer install`)
2. Configure application in dotenv file
3. Migrate and seed the database (`php artisan migrate --seed`)
4. Create storage public symlink (`php artisan storage:link`)
5. Test functionalities with `lab:*` artisan commands.

## Notes
* Default admin account is `admin:admin`.
